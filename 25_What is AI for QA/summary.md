# What is AI for QA
1. Kegunaan AI untuk Software Quality Assurance seperti Test Automation, defect prediction, code analysis dan debugging

2. AI untuk test automation tools seperti Testim, Applitools, Mabl dan Functionize, sedangkan AI untuk code analysis tools seperti CodeSonar dan Snyk.

3. Tantangan AI untuk Software Quality Assurance seperti kualitas data dan potensi bias, masalah keamanan dan privasi, kurangnya pengawasan, dan masalah transparansi.
